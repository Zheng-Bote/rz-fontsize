# Installation

```bash
npm install
npn run build
```

- copy content of folder `dist/*` into your web folder
- add `<script type="module" src="<path/to/>rz-fontsize.js"></script>` to your html-header
- add `<rz-fontsize />` to the body of your html-file (don't forget the attributes).
